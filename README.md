# C-Projects

A collection of practical C programming projects demonstrating the implementation of Data Structures and Algorithms (DSA), problem-solving techniques, and core programming concepts.

These projects are designed as beginner to intermediate level applications that combine theoretical DSA concepts with real-world use cases.
# Projects Included

## 1. URL Shortener (`url_shortener.c`)

A menu-driven URL shortening system implemented in C using Base62 encoding.

### Features
- Shorten long URLs
- Retrieve original URLs
- Base62 short code generation
- Menu-driven interface

### Concepts Used
- Structures
- Arrays
- Strings
- Functions
- Hashing logic
- Base62 Encoding


## 2. Student Management System (`student_management.c`)

A simple student record management application.

### Features
- Add student records
- Display records
- Search student by ID
- Update and delete records

### Concepts Used
- Structures
- Arrays
- Searching
- Functions
- File handling basics


## 3. Library Management System (`library_management.c`)

A basic library management application to manage books and records.

### Features
- Add books
- Issue books
- Return books
- Display available books

### Concepts Used
- Structures
- Arrays
- Searching
- Conditional statements
- Functions


## 4. Food Order Management System (`food_order_system.c`)

An online food ordering simulation using queue operations.

### Features
- Place food orders
- Serve orders using FIFO logic
- Search orders by ID
- Sort orders by price
- Display pending orders

### Concepts Used

- Structures
- Sorting
- Searching
- Arrays
- Functions
